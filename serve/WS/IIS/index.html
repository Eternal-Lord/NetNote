# Web 服务器（IIS）

TIP

IIS 是 Internet Information Services 的缩写，是一个 World Wide Web server。 Gopher server 和 FTP server 全部包容在里面。 IIS 意味着能发布网页，并且有 ASP（Active Server Pages）、JAVA、VBscript 产生页面，有着一些扩展功能。
