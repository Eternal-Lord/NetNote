# mermaid 语法

该项目通过 vuepress-plugin-mermaidjsopen in new window 插件来支持使用 mermaid-jsopen in new window（GitHubopen in new window）绘制各种图表。

支持：

  • 流程图open in new window
  • 序列图open in new window
  • 类图open in new window
  • 状态图open in new window
  • 实体关系图open in new window
  • 用户旅程图open in new window
  • 甘特图open in new window
  • 指令图open in new window
  • 饼图open in new window
